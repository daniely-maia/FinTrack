# 📊 FinTrack
**FinTrack** is a **simple console-based** personal finance tracking system, developed and improved during a **Java qualification course**.

## 🗂️ Project Structure

- FinTrack/
  - src/
    - app/
      - Main.java                 # Application entry point
    - models/
      - Transaction.java          # Base transaction class
      - MonthlyTransaction.java   # Polymorphism example
    - services/
      - FinTracker.java           # Core business logic
    - utils/
      - Formatter.java            # Currency formatting
  - CHANGELOG.md
  - LICENSE
  - README.md

## ✅ Current Features
- Add income/expense transactions
- List all transactions
- View balance
- Remove transactions by index

## 📌 Planned Features

- **Persistent Storage** – Save transactions to a database (PostgreSQL)
- **Charts and Reports** – Visualize spending patterns
- **API Integration** – Fetch exchange rates automatically
- **User Authentication** – Multi-user support
- **Graphical Interface**
